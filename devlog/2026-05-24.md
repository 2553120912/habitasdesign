### Bug 修：底部视频泄漏
- [x] 用户反馈：滚到页面最底部时，footer 区域显示出 hero 的视频画面。
- [x] 根因：`.footer` 有 `background: #1a1a1a` 但没有 `position` 也没有 `z-index`。CSS 绘制规则下，定位元素（hero `position: sticky; z-index: 1`）的绘制层级**高于**未定位的块元素，所以视频"穿透"到 footer 显示。
- [x] 修复：`.footer` 加 `position: relative; z-index: 2;`。
- [x] 验证：其他章节 `.service-process` / `.story-band` / `.works-feature` / `.about-feature` / `.contact-feature` 都已正确设置定位与 z-index，未受影响。
