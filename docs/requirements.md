# 开发需求

## 项目背景

HB 建筑事务所是一家小型建筑设计事务所，需要一个简洁高级的展示型官方网站，用于展示作品、传达设计理念、提供联系方式。

## 目标用户

- 潜在客户（业主、开发商）
- 同行 / 合作方
- 求职者 / 媒体

## 当前功能需求

### V1.0 — 基础展示网站（进行中）

- [x] 事务所名称 + 品牌标语首屏
- [x] 作品/项目展示区（占位卡片）
- [x] 关于我们简介
- [x] 联系方式
- [x] 顶部固定导航 + 平滑滚动
- [x] 响应式布局（桌面 + 手机）
- [ ] 项目详情页（点击卡片进入）
- [ ] 真实图片替换
- [ ] 事务所实际信息填充

### 待讨论

- 是否需要多语言（中文 / 英文）
- 是否需要项目分类筛选
- 是否需要新闻 / 动态板块
- 是否需要团队介绍页面

## 非功能需求

- 加载速度快（无外部依赖）
- 移动端体验良好
- 维护简单（纯静态文件，无需 CMS）
- 浏览动线清晰，用户能快速找到关键信息
